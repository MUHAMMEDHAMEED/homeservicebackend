# 🛠️ HomeService Connect

A full-stack MERN application that connects users with skilled local professionals for home services (Plumbers, Electricians, Mechanics, etc.). Built as a comprehensive college mini-project demonstrating role-based access, real-time database filtering, and cloud deployment.

**🚀 Live Demo:** [https://homeserviceconnect.netlify.app](https://homeserviceconnect.netlify.app)

## 💻 Tech Stack
* **Frontend:** React.js (Vite), CSS, Netlify (Hosting)
* **Backend:** Node.js, Express.js, Render (Hosting)
* **Database:** MongoDB Atlas (Cloud)
* **Authentication:** JWT / Google OAuth

## ✨ Key Features
* **Role-Based Dashboards:** Separate experiences and interfaces for Clients and Service Workers.
* **Strict Job Filtering:** Workers only see job requests that match their specific registered skill category.
* **Booking System:** Clients can book services with specific dates and locations.
* **Order Management:** Workers can accept pending jobs and mark active jobs as completed.
